# Images

Place photos here with these exact filenames:

| Filename              | Used for                        | Notes                                      |
|-----------------------|---------------------------------|--------------------------------------------|
| `choir-hero.jpg`      | Hero section background         | Landscape/wide shot works best. Will be darkened by CSS overlay. |
| `choir-performance.jpg` | "Our Story" section           | Any choir performance or group photo.      |
| `joe-stuligross.jpg`  | Artistic Director bio photo     | Portrait orientation preferred (3:4 ratio).|

## Tips
- JPG or PNG both work fine.
- Photos don't need to be resized — the browser handles it.
- If an image is missing, the page shows a placeholder gracefully.
